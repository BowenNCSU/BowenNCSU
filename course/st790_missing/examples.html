SAS and R Examples


We will use both R and SAS to implement many of the methods discussed in the course. Below are links to R and SAS documentation and example programs discussed in class.
  • Information on R. The R Homepage has links to manuals and other information.

  • SAS documentation. SAS 9.4 Documentation Homepage.

  • Chapter 1, EXAMPLE 1, Estimation of a single mean. R program to carry out a simulation study of estimators for a single mean under different missingness mechanisms.

  • Chapter 3, EXAMPLE 2, Maximum likelihood for bivariate normal data via the EM algorithm and direct maximization of the observed data likelihood. Homegrown R program to carry out the EM algorithm under the assumption of bivariate normal data as in EXAMPLE 2 of Chapter 3 of the notes, and output. SAS program using proc mi and proc mixed to implement the same EM algorithm using the EM statement and to maximize the observed data lklihood, simulated data set with missing values indicated using the SAS convention ".", and output. R program implementing the EM algorithm using the norm (Analysis of multivariate normal datasets with missing values) package available on CRAN, R program using the gls() function of the nlme package to do ML, and same simulated data set with missing values indicated using the R convention "NA".

  • Documentation for the R norm package.

  • Chapter 4, Implementing proper multiple imputation in SAS and R. We use the same simulated bivariate normal data sets as for the EM algorithm above. SAS program using proc mi on the simulated data set with missing values indicated using the SAS convention ".", and output. R program using the norm package and same simulated data set with missing values indicated using the R convention "NA".

  • Journal of Statistical Software article for the R package mice.

  • Chapter 4, Implementing Multivariate Imputation via Chained Equations (Fully Conditional Specification) in SAS and R. We demonstrate using a data set from a study of hypertension that contains continuous, binary, and categorical variables (see the comments in the SAS program for details). SAS program using proc mi on the data set with missing values indicated using the SAS convention ".", and output. R program using the mice package and same simulated data set with missing values indicated using the R convention "NA".

  • Chapter 5, Implementing weighted GEE analyses using proc gee in SAS. We demonstrate using a (different) simulated data set from a study of hypertension that contains continuous, binary, and categorical variables and involves only monotone missingness (dropout). SAS program using proc gee on the data set with missing values indicated using the SAS convention ".", and output. Because these are simulated data, we also have access to the full data set with no missing data; these data are also analyzed in the program for comparison to the weighted GEE analyses.
